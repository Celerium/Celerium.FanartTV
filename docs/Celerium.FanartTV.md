# Celerium.FanartTV Module
## Description
This PowerShell module acts as a wrapper for the FanartTV API.

## Celerium.FanartTV Cmdlets
### [Add-FanartTVAPIKey](site/Internal/Add-FanartTVAPIKey.md)
Sets your API key used to authenticate all API calls

### [Add-FanartTVBaseURI](site/Internal/Add-FanartTVBaseURI.md)
Sets the base URI for the FanartTV API connection

### [ConvertTo-FanartTVQueryString](site/Internal/ConvertTo-FanartTVQueryString.md)
Converts uri filter parameters

### [Export-FanartTVModuleSettings](site/Internal/Export-FanartTVModuleSettings.md)
Exports the FanartTV BaseURI, API, & JSON configuration information to file

### [Get-FanartTVAPIKey](site/Internal/Get-FanartTVAPIKey.md)
Gets the FanartTV API key

### [Get-FanartTVBaseURI](site/Internal/Get-FanartTVBaseURI.md)
Shows the FanartTV base URI

### [Get-FanartTVLatest](site/Latest/Get-FanartTVLatest.md)
Retrieve the latest fanart images for a specific resource

### [Get-FanartTVModuleSettings](site/Internal/Get-FanartTVModuleSettings.md)
Gets the saved FanartTV configuration settings

### [Get-FanartTVMovie](site/Movies/Get-FanartTVMovie.md)
Retrieve all available fanart images for a specific movie

### [Get-FanartTVMusic](site/Music/Get-FanartTVMusic.md)
Retrieve all available music artist and album images

### [Get-FanartTVShow](site/TV Shows/Get-FanartTVShow.md)
Retrieve all available fanart images for a specific TV Show

### [Import-FanartTVModuleSettings](site/Internal/Import-FanartTVModuleSettings.md)
Imports the FanartTV BaseURI, API, & JSON configuration information to the current session

### [Invoke-FanartTVRequest](site/Internal/Invoke-FanartTVRequest.md)
Makes an API request to FanartTV

### [Remove-FanartTVAPIKey](site/Internal/Remove-FanartTVAPIKey.md)
Removes the FanartTV API key

### [Remove-FanartTVBaseURI](site/Internal/Remove-FanartTVBaseURI.md)
Removes the FanartTV base URI global variable

### [Remove-FanartTVModuleSettings](site/Internal/Remove-FanartTVModuleSettings.md)
Removes the stored FanartTV configuration folder
